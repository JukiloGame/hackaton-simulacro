# hackaton-simulacro
Simulacro del dia del hackatón. Aqui tendremos el repositorio con el resultado eseprado para ese dia además de la organización inicial. Luego pasaremos todo lo que nos dejen tener para ese dia a otro repositorio

## Herramientas que llevar
### .Net (Con comandos)
SDK	.NET SDK	8.0.416
```bash
dotnet add package Microsoft.EntityFrameworkCore --version 8.0.0
dotnet add package Microsoft.EntityFrameworkCore.Sqlite --version 8.0.0
dotnet add package Microsoft.EntityFrameworkCore.Tools --version 8.0.0
dotnet add package Microsoft.EntityFrameworkCore.Design --version 8.0.0
dotnet tool install --global dotnet-ef --version 8.0.0
```
